O objetivo deste projeto é elaborar uma aplicação móvel com tecnologia WEB;

  • HTML5
  • CSS3
  • JavaScript (Ecma Script 6)
  • Bootstrap 4
  • Apache Cordova
  • AJAX (Consumir API)
  • GIT

Bem como consumir serviço (API) disponíveis na Internet

Projeto individual

Vamos criar uma aplicação WEB para dispositivo móvel e empacota-lo com Apache Cordova para smartphone (opcional).

O projeto pode ser feito com o editor de texto (Brackets ou MS VS Code), e com o Browser.

Critérios de avaliação:

  • Consumir API REST/JSON com JavaScript (AJAX)
  • Criar elementos HTML com base nos dados da API e exibir na página HTML
  • Coragem (Ref.: Extreme Programming Explained: Embrace Change (Inglês) Capa Comum – 31 out 2004 por Kent Beck, Cynthia Andres)

Entrega:

Informar no Moodle o link do projeto no GitHub com o código fonte do programa!

Esta aplicação fará parte do portfólio do aluno.
